# 🐎 競馬演出スコアラー

## 🎯 このプロジェクトについて

このアプリは、**React + Django + PostgreSQL** のフルスタック構成を学習するために作成したポートフォリオです。

| 目的           | 内容                                             |
| -------------- | ------------------------------------------------ |
| フロントエンド | React / Vite / Tailwind CSS によるSPA構成の理解  |
| バックエンド   | Django による REST API 設計・実装の理解          |
| DB             | PostgreSQL / Django ORM によるデータ永続化の理解 |
| 連携           | フロントエンドとバックエンドの API 通信の理解    |
| デプロイ       | Vercel + Render による本番環境構築の理解         |

今後は本プロジェクトで得た知識を活かし、**CSV を使った業務の日次・月次処理ツール**や**集計レポートの自動化**など、実務で活用できるアプリケーション開発に発展させていく予定です。

## 📋 機能概要

Excel解答ファイルのアップロードまたはWeb上での解答入力により自動採点するWebアプリです。
採点結果をランク・正答率・動画演出で表示し、履歴管理・CSVエクスポートにも対応しています。

## 🚀 使い方

### 採点モードの選択

トップページでExcel採点とWeb解答の2つのモードを切り替えられます。

---

### 📁 Excel採点モード

#### Excel ファイルの準備

採点には以下の2つの Excel ファイルが必要です。
サンプルファイルは `data/` フォルダに入っています。

| ファイル          | 説明                                 |
| ----------------- | ------------------------------------ |
| `answer_key.xlsx` | 正解マスタ（試験名・正解を記入）     |
| `sample.xlsx`     | ユーザー解答（受験者名・解答を記入） |

#### Excel ファイルの仕様

| セル   | 内容                                            |
| ------ | ----------------------------------------------- |
| B13    | 試験名（正解マスタ・ユーザー解答共通）          |
| B14    | 受験者名（ユーザー解答ファイルのみ）            |
| 解答欄 | 問題番号・解答を2列ペアで記入（問題数制限なし） |

> 正解マスタとユーザー解答の試験名（B13）が異なる場合はアラートでエラーを表示します。

#### 採点手順

1. **合格ライン**を設定（デフォルト70%）
2. **正解マスタ**（`answer_key.xlsx`）をアップロード
3. **ユーザー解答**（`sample.xlsx`）をアップロード
4. **採点スタート**ボタンを押す

---

### 🌐 Web解答モード

ブラウザ上で直接解答して採点できます。

#### 正解マスタの選択方法

| モード              | 説明                                           |
| ------------------- | ---------------------------------------------- |
| 📁 毎回アップロード | 正解マスタExcelをその都度アップロードして採点  |
| 🗄️ 登録済みを使う   | 管理画面で事前登録した正解マスタを選択して採点 |

#### Web解答手順

1. **受験者名**を入力
2. **正解マスタモード**を選択
3. **試験名・問題数・選択肢タイプ**を設定（アップロードモードの場合）
4. **解答をはじめる**をクリック
5. 各問題の選択肢ボタンをクリックして解答
6. 全問解答後に**採点スタート**を押す

> 選択肢は試験ごとに1〜5（数字）またはA〜E（アルファベット）に切り替え可能です。

#### 管理画面での試験登録

試験・正解を事前登録する場合は管理画面を使用してください：
`https://react-scoring-backend.onrender.com/admin/`

---

### 採点結果の表示

両モード共通で以下が表示されます：

- スコア・正答率・合格/不合格判定
- ランク（S / A / B / C）
- 動画演出
- 採点詳細（問題ごとの〇✖を2列レイアウト）
- 正答率推移グラフ

### 履歴管理

- **ユーザーフィルター** — 受験者名で絞り込み
- **試験名フィルター** — 試験名で絞り込み
- **CSVダウンロード** — 採点履歴を CSV でエクスポート
- **削除ボタン** — 不要な履歴を削除（削除ボタンクリック時のみ）

> 履歴はRender PostgreSQLに永続保存されます。再デプロイしても履歴は消えません。

## 📊 主な機能

- **Excel自動採点** — Excelファイルを比較して正誤判定（問題数制限なし）
- **Web解答採点** — ブラウザ上で解答して即採点
- **正解マスタDB登録** — 管理画面から正解マスタを登録・使い回し可能
- **試験整合性チェック** — 異なる試験の組み合わせをアラートで検出
- **合格ライン設定** — 採点前に合格ラインを設定（デフォルト70%）
- **合格・不合格判定** — 設定した合格ラインに基づき合否を表示
- **ランク判定** — S / A / B / C の4段階
- **動画演出** — スコアに応じた動画をランダム再生
- **採点詳細** — 問題ごとの〇✖を2列レイアウトで表示
- **正答率推移グラフ** — 受験者ごとの推移を折れ線グラフで表示
- **履歴管理** — ユーザー・試験名でフィルタリング可能
- **CSV エクスポート** — 採点履歴を CSV でダウンロード
- **履歴永続化** — PostgreSQL により再デプロイ後も履歴を保持

## 🔌 API エンドポイント

| メソッド | URL                          | 説明                 |
| -------- | ---------------------------- | -------------------- |
| POST     | `/api/score/`                | Excel採点実行        |
| POST     | `/api/score/web/`            | Web解答採点実行      |
| GET      | `/api/history/`              | 履歴一覧取得         |
| DELETE   | `/api/history/delete/<id>/`  | 履歴削除             |
| GET      | `/api/history/export/`       | CSV エクスポート     |
| GET      | `/api/exams/`                | 試験一覧取得         |
| GET      | `/api/exams/<id>/questions/` | 問題一覧取得         |
| POST     | `/api/exams/<id>/submit/`    | 登録済みマスタで採点 |

## 🐎 ランク基準

| ランク | 正答率   |
| ------ | -------- |
| S      | 100%     |
| A      | 70% 以上 |
| B      | 50% 以上 |
| C      | 50% 未満 |
